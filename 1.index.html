Read and set simple value from localStorage

Why Local Storage is used in browser?

Local Storage is used to smooth user experience.

সাধারনত e commerce website গুলাতে কোন কিছু cart এ add করলে অই website টা refresh করার পরেও cart এ product add হয়ে থাকে যা browser এর local storage এর কারণে।

browser inspect করলে Application এ click করলে key আর value দেখা যায়। এই key, value টা কিছুটা variable এর মত। এখানে key হচ্ছে variable name এবং value হচ্ছে varialble এর value।

ধরা যাক localStorage কোন data manually save করে রাখা আছে। যদি সেই data access করা হয় তাহলে কিছু method আছে। see the ex-1 for getItem. Note: localStorage এ সব data string অবস্থায় থাকে। see the ex-2 for setItem, getItem & removeItem.
